# 🏨 Hotel Management System

A console-based Hotel Management System developed using C++ that efficiently manages hotel operations such as room booking, customer records, and billing.
The project demonstrates the use of data structures and object-oriented programming to handle real-world scenarios.

---

## 🚀 Features
- Room booking and reservation management
- Customer record storage and retrieval
- Check-in and check-out functionality
- Billing and payment calculation
- Search and update customer details

---

## 🛠️ Tech Stack
- C++
- Data Structures (Arrays, Linked Lists)
- Object-Oriented Programming (OOP)
- File Handling

---

## 📌 How It Works
- Users can book rooms and store customer details
- System manages records using data structures
- Allows searching, updating, and deleting records
- Handles billing during check-out

---

## 💡 Concepts Used
- Data Structures (Arrays, Linked Lists, Maps)
- Searching and Sorting
- File Handling
- Object-Oriented Programming
